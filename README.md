# 🎯 MVC Number Guessing Game

## 📌 Project Overview

MVC Number Guessing Game is a web-based game developed using **ASP.NET MVC** where players try to guess a randomly generated number within limited attempts.

The system provides instant feedback, tracks attempts, and improves logical thinking through interactive gameplay.

---

## 🚀 Features

* Random number generation
* Attempt counter
* Success & failure messages
* MVC Architecture implementation
* Responsive UI
* Session-based game logic

---

## 🛠️ Technologies Used

* ASP.NET MVC
* C#
* HTML
* CSS
* JavaScript
* Visual Studio 2022

---

## 🎮 How to Run the Project

1. Clone the repository
2. Open solution in Visual Studio 2022
3. Restore NuGet packages
4. Run using IIS Express
